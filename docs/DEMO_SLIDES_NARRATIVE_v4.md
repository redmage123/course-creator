## Video Production Requirements

### Technical Specifications:
- **Resolution:** 1920x1080 (Full HD)
- **Frame Rate:** 30 fps
- **Format:** H.264 MP4 (baseline profile, level 3.0)
- **Bitrate:** ~2-3 Mbps (target file size: 5-15 MB per slide)
- **Audio:** Separate audio file (will be synced in demo player)
- **Aspect Ratio:** 16:9

### Recording Environment:
- Platform: Course Creator Platform (staging/demo environment)
- Browser: Chrome/Firefox with developer tools
- Screen Recording: OBS Studio or similar
- Cursor Highlighting: Custom cursor overlay
- Text Annotations: Added in post-production if needed

### Quality Standards:
- Clear, smooth mouse movements
- No errors or typos in forms
- Consistent timing with narration
- Professional pacing (not rushed)
- Clean transitions between sections
- No loading spinners longer than 2 seconds (edit out if needed)
