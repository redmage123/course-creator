## Post 6 — Docker Cost Deep-Dive

The math on Docker labs vs cloud VMs for education:

Cloud VM (t3.small) per learner per hour: ~€0.02–0.05
Docker container per learner: ~€0.001–0.003

At 100 learners, 10-hour course, Docker costs roughly 15–20x less.

For bootcamps running 50+ students per cohort, that difference is real money.
